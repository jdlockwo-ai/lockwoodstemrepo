Teacher setup

Google Sheet format

Use the spreadsheet template to make quick agenda edits without changing the website code every day.

1

Fill out the sheet

Use one tab per course. Keep the column headers the same so the website can read the data.

2

Publish each tab

In Google Sheets, publish each course tab as a CSV link using File → Share → Publish to web.

3

Update one config file

Paste the published CSV links into assets/js/agenda-config.js. Future sheet edits will display on this page.
