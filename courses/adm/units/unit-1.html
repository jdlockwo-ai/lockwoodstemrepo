ADM Unit 1

Introduction to Industry 4.0

Students explore smart manufacturing through robotic arm movement, assembly lines, pick-and-place programming, material handling, palletizing, and a packaging challenge.

View Lessons Unit Challenge Back to ADM
Unit Goal

From motion to manufacturing workflow

Learn how robotic systems move, repeat tasks, organize products, and support packaging and shipping operations.

Essential Questions

  • What makes a manufacturing system smart?
  • How do robotic arms support repeatable manufacturing tasks?
  • How do programming decisions affect accuracy, safety, and efficiency?
  • How do logistics tasks connect to Industry 4.0?

Learning Targets

  • I can describe how robots are used in modern manufacturing.
  • I can program or analyze robotic movement sequences.
  • I can compare absolute and relative movement.
  • I can document setup, testing, code, and revisions.

Major Deliverables

  • Engineering notebook entries
  • Robot program or code documentation
  • Testing and troubleshooting evidence
  • Pack and Ship Challenge reflection
Lesson sequence

Unit 1 lesson clusters

The unit moves from basic robotic arm motion to manufacturing logistics and a culminating packaging challenge.

1.1

Robotic Arm Fundamentals & Motion Programming

Students learn how robotic arms move, follow paths, complete pick-and-place tasks, and use absolute and relative movement.

Activity 1.1.1Assembly Line

Investigate how assembly lines organize tasks and improve consistency, efficiency, and repeatability.

Activity 1.1.26-Axis Arm

Explore joints, axes of motion, reach, and the work envelope of a robotic arm.

Activity 1.1.3Pick and Place

Apply precise movement to transport an object from one location to another.

Activity 1.1.4Coding Movement

Practice simple motion commands and movement sequences.

Activity 1.1.5Controlled Path

Analyze how path planning improves safety, accuracy, and efficiency.

Activity 1.1.6Shapes

Use code to move through geometric paths and connect programming to spatial reasoning.

Activity 1.1.7Absolute and Relative Movement

Compare absolute and relative movement commands and when each is useful.

Project 1.1.8Decreasing Spirals

Create a programmed robotic motion sequence using path control and geometric reasoning.

1.2

Manufacturing Careers, Material Handling & Palletizing

Students connect Industry 4.0 skills to careers and apply robotic movement to transporting, palletizing, and stacking objects.

Activity 1.2.1Career Exploration

Investigate careers in advanced manufacturing, robotics, automation, logistics, and engineering technology.

Activity 1.2.2Transporting and Palletizing Objects

Explore how automated systems move, organize, and palletize objects in manufacturing environments.

Project 1.2.3Stacking Objects

Apply movement, sequencing, and spatial planning to stack objects accurately and repeatably.

1.3

Unit Challenge Problem

Students combine robotic movement, material handling, workflow planning, testing, and documentation in a realistic packaging task.

Problem 1.3.1Pack and Ship Challenge

Create a repeatable packaging or shipping process that organizes, moves, stacks, packs, or prepares objects for delivery.
