Lesson Focus

Code, test, document

Use the example as a starting point, then document what the code does and how it was tested.

Lesson Overview

In this lesson, students practice using code to move the robotic arm through a sequence of points. The goal is to connect each movement command to a physical location and observe how ordered commands create repeatable motion.

Notes

  • Run the arm from a known safe starting position.
  • Read each movement command as a coordinate target.
  • Pay attention to how the final value changes the arm height or tool position.
  • Use testing evidence to explain what the robot does at each step.
  • Add meaningful comments before submitting code.
