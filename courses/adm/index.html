Course Focus

Smart manufacturing systems

Students learn how robotics, sensors, data, simulation, logistics, cybersecurity, and documentation work together in modern production.

Course pathway

ADM Units

Each unit page includes an overview, essential questions, learning targets, lesson sequence, challenge, deliverables, certifications, resources, and vocabulary.

Unit 1

Introduction to Industry 4.0

Explore smart manufacturing, robotic arm movement, assembly lines, material handling, palletizing, and packaging workflows.

 Unit 2

Advanced Automation in Manufacturing

Build on foundational robotics by using sensors, programmed decisions, workcells, pneumatics, and automated process control.

 Unit 3

Digital Systems and Virtual Integration

Use digital twins, CAD/CAM workflows, motion studies, simulation, and virtual manufacturing tools to test before building.

 Unit 4

Securing the Supply Chain

Study cybersecurity, supplier decisions, packaging, logistics, quality control, risk, and resilient manufacturing systems.
Course habits

How students work in ADM

ADM connects technical skills with professional engineering documentation.

Design

Plan the system

Sketch layouts, map workflows, identify constraints, and prepare files before building.

Automate

Program behavior

Use movement commands, sensors, logic, and testing to control manufacturing tasks.

Inspect

Use evidence

Measure results, document failures, compare outcomes, and revise based on data.

Communicate

Document the process

Maintain notebooks, code comments, photos, test logs, and final reflections.
